    **Algorithm (in this exact order):**

    1. Parse argv. Resolve `--label` (default `'baseline'`), `--out` (default `join(REPO_ROOT, '.planning', 'intel', 'vitest-integration-baseline.md')`), `--append` (boolean).
    2. Compute `TMP_DIR = join(REPO_ROOT, 'sdk', 'scripts', '.tmp', 'profile-integration')`. `mkdirSync(TMP_DIR, { recursive: true })`. Wipe stale `run-*.json` from prior invocations: `for (const i of [1,2,3]) { const p = join(TMP_DIR, \`run-${i}.json\`); if (existsSync(p)) rmSync(p); }`.
    3. **Build once before the loop** (per RESEARCH "What `pretest` actually costs" + PATTERNS.md `pnpm --filter` invocation pattern — isolates test wall-clock from rebuild cost):
       ```js
       console.error('[profile-integration] building sdk once before measurement loop...');
       execFileSync('pnpm', ['--filter', '@gsd-build/sdk', 'run', 'build'], {
       	cwd: REPO_ROOT, stdio: 'inherit',
       });
       ```
    4. **Median-of-3 loop:**
       ```js
       const runs = [];
       for (const i of [1, 2, 3]) {
       	const outFile = join(TMP_DIR, `run-${i}.json`);
       	const startedAt = Date.now();
       	console.error(`[profile-integration] run ${i}/3 starting...`);
       	try {
       		execFileSync('pnpm', [
       			'--filter', '@gsd-build/sdk', 'exec',
       			'vitest', 'run',
       			'--project', 'integration',
       			'--reporter=json',
       			'--outputFile', outFile,
       		], { cwd: REPO_ROOT, stdio: ['ignore', 'inherit', 'inherit'] });
       	} catch (err) {
       		// vitest exits non-zero on failure; we still want to parse the JSON and report which test failed.
       		console.error(`[profile-integration] run ${i} exited non-zero — parsing JSON for diagnostics`);
       	}
       	const wall = Date.now() - startedAt;
       	const parsed = JSON.parse(readFileSync(outFile, 'utf-8'));
       	runs.push({ parsed, wallClockMs: wall });
       }
       ```
    5. **D-05c flakiness gate** (per PATTERNS.md "Diagnostic-exit gate" + check-skip-count.cjs:108-121 shape):

       const failedRuns = runs.filter((r, idx) => !r.parsed.success).map((_, idx) => idx + 1);

       }
