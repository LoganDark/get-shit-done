  "build:cjs": "tsc -p tsconfig.cjs.json",
  "check:alias-drift": "pnpm run build && node scripts/check-command-aliases-fresh.mjs",

- Do NOT add `concurrently` as a dev dependency (RESEARCH Pitfall: avoid heavy npm deps).
